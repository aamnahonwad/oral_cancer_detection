OralScan AI – Oral Cancer Detection System

Developed by Aamna

Project Overview

OralScan AI is a web-based application designed to assist in the early detection of oral cancer using image analysis. The system uses a trained deep learning model to classify oral cavity images as either normal or cancerous, along with a confidence score and visual explanation.

Features

AI-based oral cancer detection
Confidence score prediction
Heatmap visualization using Grad-CAM
Image upload and analysis
Report generation
User profile and history tracking
Responsive user interface

Tech Stack

Frontend: HTML, CSS
Backend: Python (Flask)
AI Model: Convolutional Neural Network (CNN) / MobileNetV2
Libraries: TensorFlow, OpenCV, NumPy, Matplotlib
Database: MySQL

Model Performance
 Training Accuracy: approximately 92%
 Validation Accuracy: approximately 93.56%




Working

The user uploads an oral image
The image is preprocessed and passed to the trained model
The model predicts:
Normal or Cancer
Confidence score
Grad-CAM generates a heatmap highlighting important regions
